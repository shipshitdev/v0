## CLI Flow

Interactive flow:

1. Detect installed `codex` and `claude`.
2. Ask agent first.
3. Ask project scope.
4. Ask app surfaces with checkbox selector.
5. Ask routes with checkbox selector.
6. Ask optional GitHub setup with `gh`; default is no.
7. Render a progress task board.
8. Scaffold files.
9. Run selected agent unless skipped.
10. Optionally create GitHub repo/issue.
11. Run `bun install` unless skipped.
12. Print selected app scripts.
13. Start `apps/web` when selected unless skipped.

Non-interactive flags:

- `--scope`
- `--agent claude|codex`
- `--apps web,app,desktop,mobile,extension`
- `--routes overview,new-task,search,inbox,activities`
- `--github`
- `--github-repo owner/name`
- `--github-issue`
- `--github-visibility private|public`
- `--skip-agent`
- `--no-install`
- `--no-start`
- `--yes`
